How to Enable Microphone Permissions

Step 1: Check the address bar for permission icons
When you click "Start Tracking" in the extension, Chrome will ask for microphone permission. Look for the microphone icon in the address bar or the extension popup - it indicates the permission status.
If Chrome isn't showing the permission dialog at all, this could mean permissions were previously denied.
Step 2: Grant permissions for the extension popup

For extension popups: Click on the lock/site settings icon in the address bar. Look for "Microphone" in the dropdown menu and select "Allow".

For Chrome extension: The URL of the extension popup will look something like "chrome-extension://[extension-id]/popup.html"

Step 3: Check Chrome's site settings

If you can't find the permission dialog or the above steps don't work:

  1. Open Chrome Settings (three dots menu → Settings)
  2. Go to "Privacy and Security" → "Site Settings" → "Microphone"
  3. Look for entries that start with "chrome-extension://" and ensure they're set to Allow
  4. If you see any entries set to Block, click on them and change to Allow
Step 4: Reset the extension if needed

Close and reopen the extension popup. Then try clicking "Start Tracking" again.

Note: If you don't see the permission dialog, you might need to check your Chrome settings by going to:

Chrome Settings → Privacy and Security → Site Settings → Microphone

Look for entries related to the extension popup and make sure they're set to "Allow".
Common Issues & Solutions

1. "Permission Denied" message: You previously denied microphone access. Click the lock/info icon in the address bar to change permission settings.

2. No permission dialog appears: Try reloading the extension completely from chrome://extensions (toggle the extension off and on).

3. Extension doesn't work after allowing permissions: Close and reopen the extension popup.
